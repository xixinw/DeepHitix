# Attribution Policy

This project separates third-party relationship into three categories:

1. Directly derived code
   - File header required.
   - Original copyright and license preserved.

2. Adapted code
   - File header required.
   - Adaptation notes preferred.

3. Inspired architecture / ideas
   - Mention in architecture docs and THIRD_PARTY_NOTICES.
   - File-level copyright is not required unless code is copied.

Current status at initiation:
- Hermes Agent: planned base/reference, MIT.
- DeepSeek-Reasonix: idea inspiration only, MIT.
