# Privacy Policy

DeepSeek Native Agent is designed as a self-hosted application.

Default privacy posture:
- No project-operated telemetry server.
- API keys remain on the user's machine or server.
- Cost telemetry is stored locally.
- Prompt/response bodies are not written to cost telemetry by default.
- Users must be able to clear local logs, memory, and cost records.

This document must be revised before any hosted service or remote telemetry feature is introduced.
